# uTools 年会抽奖插件

一个功能完善的年会抽奖工具,支持以下功能:

- 配置抽奖总人数和奖项
- 导入名单和照片
- 多种抽奖模式
- 抽奖结果保存
- 重置功能
- 背景音乐

## 使用方法

1. 在 uTools 中输入 "抽奖"/"年会抽奖" 等关键词启动插件
2. 点击"抽奖配置"设置总人数和奖项
3. 可选择导入名单或照片
4. 点击"开始"按钮开始抽奖
5. 点击"停止"按钮停止抽奖并显示结果

## 注意事项

- 建议在抽奖前提前导入名单或照片
- 抽奖过程中可以开启/关闭背景音乐
- 抽奖结果会自动保存,可在"抽奖结果"中查看
